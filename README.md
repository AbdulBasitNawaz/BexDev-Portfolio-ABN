# BexDev Portfolio
## Welcome to my personal developer portfolio!
This portfolio showcases my skills, projects, and experience as a full-stack developer, with a focus on modern web technologies and clean UI/UX.

## This project is built with:
* React.js
* Vite
* Tailwind CSS
* Framer Motion (if you used animations)
* JavaScript & JSX


1. Clone the repository
* git clone https://github.com/AbdulBasitNawaz/BexDev-Portfolio-ABN.git
2. Install dependencies
* npm install
3. Start the development server
* npm run dev

## The project will run at:
For example: http://localhost:5173/
